# Récits utilisateurs ("user story")

| Fonctionnalité                                                                                                                                                                                                                                      | Complexité | Priorité |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------- | -------- |
| **En tant** que gestionnaire de mobilier, **je souhaite** stocker du mobilier non-affecté / inutilisé à un endroit précis.                                                                                                                          | 3          | 3        |
| **En tant** que gestionnaire de mobilier, **je souhaite** mettre à disposition un mobilier pour un gestionnaire de site (sortir le meuble du stockage et le rendre actif).                                                                          | 34         | 3        |
| **En tant** que gestionnaire de site, **je souhaite** être notifié lorsqu'un utilisateur lambda effectue un déplacement de mobilier.                                                                                                                | 55         | 3        |
| **En tant** qu'utilisateur de mobilier, **je souhaite** pouvoir déplacer mon mobilier (éventuellement automatiquement).                                                                                                                             | 21         | 5        |
| **En tant** que responsable de site, **je souhaite** affecter/desaffecter du mobilier à un collaborateur.                                                                                                                                           | 3          | 5        |
| **En tant** que loueur de mobilier / client, **je souhaite** visualiser l'état fonctionnel d'un mobilier en extrayant les données du Digital Passport.                                                                                              | 3          | 8        |
| **En tant** que gestionnaire de mobilier, **je souhaite** enregistrer du mobilier en plus dans mes bureaux.                                                                                                                                         | 3          | 13       |
| **En tant** que gestionnaire de mobilier, **je veux** pouvoir enregistrer et mettre à jour la position d'un mobilier de manière manuelle (selon un système de référence: batiment, étage, salle, position dans la salle).                           | 3          | 8        |
| **En tant** que gestionnaire de mobilier, **je veux** pouvoir enregistrer et mettre à jour la position d'un mobilier de manière automatique (scan d'un identifiant sur le mobilier lié à une information de localisation par GPS ou autre système). | 89         | 13       |
| **En tant** qu'utilisateur / loueur de mobilier, **je souhaite** visualiser l'historique d'une pièce de mobilier.                                                                                                                                   | 89         | 13       |
| **En tant** que gestionnaire de mobilier / gestionnaire de site, **je souhaite** pouvoir filtrer et rechercher les meubles présents (type de mobilier, zone géographique, état du mobilier, etc...).                                                | 13         | 13       |
| **En tant** qu'utilisateur, **je souhaite** exporter une liste de mobilier en fonction de critères spécifique.                                                                                                                                      | 21         | 13       |
| **En tant** que responsable de site / loueur de mobilier, **je souhaite** pouvoir visualiser l'emplacement du mobilier sur une carte (2D - Plan du batiment ? Dessins de pièces ?).                                                                 | 34         | 13       |
| **En tant** qu'administrateur, **je veux** pouvoir créer des comptes, pour donner l'accès à des personnes de l'application.                                                                                                                         | 34         | 21       |
| **En tant** qu'administrateur, **je veux** pouvoir changer les permissions/roles d'un compte.                                                                                                                                                       | 34         | 34       |
| **En tant** qu'administrateur, **je veux** pouvoir voir les comptes qui ont accès à l'application.                                                                                                                                                  | 34         | 34       |
| **En tant** que gestionnaire de mobilier, **je souhaite** me débarasser / céder du mobilier inutilisé.                                                                                                                                              | 34         | 34       |
